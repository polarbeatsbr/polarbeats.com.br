## 🎯 Sobre

A **Polar Beats** é uma inovação patenteada (INPI BR 202023011705-3) que integra climatização inverter de alta eficiência com sistema de áudio premium de alta fidelidade em um único equipamento — criando uma categoria inteiramente nova no mercado.

## ✨ Features do Site

- 🌍 **Multilíngue** — Português, English, Español
- 🎵 **Simulador de Áudio** interativo com modos ambientais
- 📊 **Calculadora B2B** de economia para distribuidores
- 📱 **Responsivo** — Otimizado para mobile, tablet e desktop
- 🎨 **Animações** suaves com scroll reveal
- 🌙 **Design dark premium** com efeitos glassmorphism

## 🛠️ Tecnologias

- HTML5
- CSS3 (Vanilla — sem frameworks)
- JavaScript (Vanilla — sem dependências)
- Google Fonts (Syne + Plus Jakarta Sans)

## 🚀 Como Rodar

Basta abrir o `index.html` no navegador ou usar um servidor local:

```bash
# Com Python
python -m http.server 8000

# Com Node.js
npx serve .
```
